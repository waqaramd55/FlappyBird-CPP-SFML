# Flappy Bird Clone - C++ & SFML 3.0

A cross-platform Flappy Bird clone built using **C++** and the **SFML 3.0** multimedia library.

##  Features
- **Dynamic Scoring**: +10 points for every pipe passed.
- **Physics**: Real-time gravity and bird "flap" mechanics.
- **State Management**: Distinct Start, Play, and Game Over states.
- **High Score**: Persistent high score tracking during the session.

##  Tech Stack
- **Language**: C++
- **Library**: SFML 3.0 (Graphics, Window, System)
- **Build System**: CMake
- **Compiler**: MSVC (Visual Studio Code)

##  How to Run
1. Ensure you have SFML 3.0 installed.
2. Use CMake to configure the project.
3. Build and run `FlappyBird.exe`.
